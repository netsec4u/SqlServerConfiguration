# Disable-SqlDatabaseMail

## SYNOPSIS

Disables Database Mail.

## SYNTAX

### ServerInstance (Default)

```
Disable-SqlDatabaseMail
  -ServerInstance <string>
  [-WhatIf]
  [-Confirm]
  [<CommonParameters>]
```

### SmoServerObject

```
Disable-SqlDatabaseMail
  -SmoServerObject <Server>
  [-WhatIf]
  [-Confirm]
  [<CommonParameters>]
```

## ALIASES

This cmdlet has the following aliases:
  None

## DESCRIPTION

Disables Database Mail on a SQL Server instance.

## EXAMPLES

### Example 1

```powershell
Disable-SqlDatabaseMail -ServerInstance MyServer
```

Disables Database Mail on the specified SQL Server instance.


### Example 2

```powershell
$SmoServer = Connect-SmoServer -ServerInstance MyServer
Disable-SqlDatabaseMail -SmoServerObject $SmoServer
```

Disables Database Mail on the specified SQL Server instance using the SmoServer session.
