# New-SqlDatabaseMailProfile

## SYNOPSIS

Creates a new Database Mail profile on a SQL Server instance.

## SYNTAX

### ServerInstance (Default)

```
New-SqlDatabaseMailProfile
  -ServerInstance <string>
  [-MailProfileName <string>]
  [-Description <string>]
  [-WhatIf]
  [-Confirm]
  [<CommonParameters>]
```

### SmoServerObject

```
New-SqlDatabaseMailProfile
  -SmoServerObject <Server>
  [-MailProfileName <string>]
  [-Description <string>]
  [-WhatIf]
  [-Confirm]
  [<CommonParameters>]
```

## ALIASES

This cmdlet has the following aliases:
  None

## DESCRIPTION

Creates a new Database Mail profile on a SQL Server instance.

## EXAMPLES

### Example 1

```powershell
New-SqlDatabaseMailProfile -ServerInstance "MyServer" -MailProfileName "SupportProfile" -Description "Profile for Support Team"
```

Creates a new Database Mail profile named "SupportProfile" on the SQL Server instance "MyServer" with the description "Profile for Support Team".

### Example 2

```powershell
$SmoServer = Connect-SmoServer -ServerInstance MyServer
New-SqlDatabaseMailProfile -SmoServerObject $SmoServer -MailProfileName "SupportProfile" -Description "Profile for Support Team"
```

Creates a new Database Mail profile named "SupportProfile" on the SQL Server instance using the SmoServer session.
